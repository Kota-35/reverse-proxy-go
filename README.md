# Reverse Proxy Server in Go

Go言語で実装されたシンプルなリバースプロキシサーバーです。複数のバックエンドサーバーへのリクエストを設定に基づいてプロキシします。

## 機能

- 🚀 設定可能なリバースプロキシ
- 📝 YAML形式の設定ファイル
- 🔍 ヘルスチェックエンドポイント
- 📊 リクエストログ出力
- 🔄 複数バックエンドサーバー対応
